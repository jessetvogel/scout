# 🏕️ scout

Scout is a Python data exploration tool.

## 📦 Installation

Scout can be installed directly using `pip`.

```sh
pip install "scout @ git+https://github.com/jessetvogel/scout.git@main"
```

Alternatively, scout can be installed by adding the following dependency to your
`pyproject.toml` file.

```toml
[project]
dependencies = [
  "scout @ git+https://github.com/jessetvogel/scout.git@main"
]
```
